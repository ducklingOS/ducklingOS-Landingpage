System Requirements

Minimum Requirements

  • Processor: 64-bit x86_64 processor
  • Memory: 2GB RAM
  • Storage: 20GB free disk space
  • Graphics: VGA compatible with 800x600 resolution

Recommended Requirements

  • Processor: Modern 64-bit processor (Intel/AMD)
  • Memory: 4GB RAM or more
  • Storage: 50GB SSD or more
  • Graphics: Modern GPU with OpenGL 3.3+ support
  • Display: 1280x720 resolution or higher

Supported Hardware

DucklingOS supports a wide range of hardware configurations. Most modern PCs and laptops should work without issues.

  • Intel and AMD processors
  • NVIDIA, AMD, and Intel graphics
  • Most Wi-Fi and Ethernet adapters
  • USB 2.0 and 3.0 devices
  • SATA and NVMe SSDs

Installation Guide

💾 USB Installation

Recommended method for most users. Creates a bootable USB drive.

  1. Download the ISO file
  2. Use a tool like Balena Etcher or Rufus
  3. Select the ISO and your USB drive
  4. Flash the image to USB
  5. Boot from USB and follow installer

💵 Virtual Machine

Test DucklingOS in a virtual environment first.

  1. Download the ISO file
  2. Create a new VM in VirtualBox/VMware
  3. Allocate at least 2GB RAM and 20GB storage
  4. Attach the ISO as boot media
  5. Start the VM and follow installer

🛠 Dual Boot Setup

Install alongside your existing operating system.

  1. Backup your important data
  2. Create free space on your hard drive
  3. Create a bootable USB
  4. Boot from USB and select installation
  5. Choose dual-boot option during setup
